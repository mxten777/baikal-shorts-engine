# 썸네일 생성 기능 가이드

> 업데이트: 2026-03-16

BAIKAL Shorts Engine의 스마트 썸네일 생성 기능 안내입니다.

---

## 📸 개요

렌더링 시 영상과 함께 **YouTube/Instagram 업로드용 썸네일 이미지**가 자동으로 생성됩니다.

### 주요 특징
- ✅ **4가지 전문적인 스타일** 제공
- ✅ **1080x1920 Full HD** 해상도
- ✅ **자동 텍스트 줄바꿈** 및 중앙 정렬
- ✅ **브랜드 로고 자동 삽입** (gradient 스타일)
- ✅ **즉시 다운로드** 가능

---

## 🎨 썸네일 스타일

### 1. 그라디언트 (Gradient) — 추천 ⭐

**특징:**
- 세련된 다크 블루 → 시안 그라디언트 배경
- 글로우(빛 번짐) 효과로 프리미엄 느낌
- 장식 요소: 원형 라인, 상단 액센트 바
- 브랜드 로고 하단 중앙 배치

**적합한 콘텐츠:**
- IT/테크 관련 콘텐츠
- 비즈니스/전문 서비스
- 대부분의 일반 콘텐츠

**미리보기:**
```
┌─────────────────────────┐
│ ━━━━━━━━━━━━━━━━━━━━━━ │ ← 액센트 바
│                         │
│    ✨ 글로우 효과 ✨     │
│                         │
│   [썸네일 텍스트]       │ ← 대형 폰트, 시안색
│                         │
│         🏔 로고          │ ← 하단 중앙
└─────────────────────────┘
```

## 🚀 사용 방법

### 1. 렌더링 설정에서 스타일 선택

프로젝트 상세 페이지에서:

1. **"렌더링 시작"** 버튼 클릭
2. 렌더 설정 다이얼로그에서 **"썸네일 스타일"** 드롭다운 확인
3. 4가지 옵션 중 선택:
   - 🌈 그라디언트 (추천)
   - ⚡ 테크 스타일
   - ✨ 미니멀
   - 💪 대담한 스타일

### 2. 썸네일 텍스트

- AI가 자동으로 생성한 **썸네일 텍스트**가 사용됩니다
- 기획 단계에서 GPT가 3가지 옵션 제안 → 첫 번째가 기본값
- 15자 이내로 간결하게 구성

### 3. 결과 확인 및 다운로드

렌더링 완료 후:

1. **렌더 결과 페이지**로 자동 이동
2. 영상 하단에 **"썸네일 이미지"** 섹션 표시
3. 미리보기 이미지 확인
4. **"다운로드"** 버튼 클릭 → `thumbnail.jpg` 저장

## ⚙️ 기술 세부사항

### 이미지 사양

| 항목 | 값 |
|------|-----|
| 해상도 | 1080 × 1920 (세로형) |
| 포맷 | JPEG |
| 품질 | 95 |
| 색상 공간 | RGB |
| 파일 크기 | 평균 200-400KB |

### 폰트

- **한글**: Noto Sans KR Bold
- **크기**: 60-100px (텍스트 길이에 따라 자동 조정)
- **줄바꿈**: 10-15자 단위 자동 줄바꿈

### 브랜드 컬러

| 색상 | Hex | 용도 |
|------|-----|------|
| 배경 (다크) | #0A1628 | 그라디언트, 테크, 대담 |
| 배경 (라이트) | #FFFFFF | 미니멀 |
| 액센트 (시안) | #00D4FF | 텍스트, 장식 요소 |
| 텍스트 (화이트) | #FFFFFF | 대부분 스타일 |
| 텍스트 (다크) | #0A1628 | 미니멀 스타일 |

```python
    img = Image.new("RGB", (w, h), (255, 0, 0))  # 빨간 배경
    draw = ImageDraw.Draw(img)
    
    # ... 디자인 로직
    
    return img
```

### 스타일 등록

1. `RenderConfig` 스키마에 새 옵션 추가:
```python
# backend/app/schemas/render.py
thumbnail_style: Literal["gradient", "tech", "minimal", "bold", "custom"]
```

2. `_create_and_upload_thumbnail` 함수에 분기 추가:
```python
elif style == "custom":
```


## ❓ FAQ

### Q: 썸네일 텍스트를 수정할 수 있나요?

A: 현재는 AI가 생성한 텍스트가 자동으로 사용됩니다. 향후 업데이트에서 수동 편집 기능이 추가될 예정입니다.

### Q: 로고를 다른 것으로 변경하려면?

A: `backend/assets/templates/baikal_logo.png` 파일을 교체하면 됩니다. PNG 형식, 투명 배경 권장.

### Q: 다른 해상도도 지원하나요?

A: 현재는 1080×1920만 지원합니다. YouTube/Instagram에 최적화된 사이즈입니다.

### Q: 썸네일 없이 렌더링할 수 있나요?

A: 썸네일은 렌더링 프로세스에 자동 포함되며, 별도로 비활성화할 수 없습니다. 단, 다운로드하지 않으면 됩니다.
